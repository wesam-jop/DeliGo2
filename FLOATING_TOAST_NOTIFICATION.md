# 🔔 نظام الإشعارات العائم (Floating Toast)

---

## ✅ كيف يعمل الآن

### **التدفق الكامل:**

```
Backend يرسل إشعار عبر ntfy
         ↓
useNtfy Hook يستقبل الإشعار (SSE)
         ↓
يعالج البيانات ويستخرج:
  - العنوان (title)
  - الرسالة (message)
  - الرابط (action_url)
  - الصوت (sound_name)
  - الأولوية (priority)
         ↓
addToast() ← يعرض الإشعار العائم
         ↓
يظهر للمستخدم إشعار عائم في أعلى اليسار
         ↓
المستخدم يضغط زر "عرض التفاصيل"
         ↓
1. markAsOpened(id) → Backend
2. handleDeepLink(action_url) → Navigate للصفحة
```

### **1. ToastNotifications.jsx** (محدث ✅)

**الميزات الجديدة:**
- ✅ **زر واضح "عرض التفاصيل"** مع أيقونة سهم
- ✅ **Progress Bar** يوضح الوقت المتبقي
- ✅ **ألوان حسب الأولوية** (أحمر للعالي، أزرق للعادي)
- ✅ **أيقونات حسب النوع** (طلب = برتقالي، رسالة = أزرق)
- ✅ **إخفاء تلقائي بعد 8 ثواني**

### **2. useNtfy.js** (محدث ✅)

**الميزات:**
- ✅ يتصل بـ ntfy عبر SSE
- ✅ يستقبل الإشعارات
- ✅ يشغل الصوت (إذا مش silent)
- ✅ يعرض الـ Toast تلقائياً
- ✅ عند الضغط على الزر → يفتح الرابط

### **3. app.jsx** (محدث ✅)

**التحديث:**
```jsx
const AppWithNtfy = () => {
    const { token, user } = useAuth();
    const navigate = useNavigate(); // ← تمت الإضافة

    useNtfy(user?.ntfy_topic, {
        enabled: !!user?.ntfy_topic,
        navigate: navigate,  // ← تمرير navigate
        playSound: true,
    });

    return <AppRoutes />;
};
```

## 🔧 التخصيص

### **تغيير مدة العرض:**

```jsx
// ToastNotifications.jsx - السطر ~30

const duration = 8000; // 8 ثواني (غيّر الرقم)
```

### **تغيير مكان الإشعار:**

```jsx
// ToastContainer - السطر ~13

<div className="fixed top-4 left-4 z-[9999] ...">
  // غيّر إلى:
  // top-4 right-4 → أعلى اليمين
  // bottom-4 left-4 → أسفل اليسار
  // bottom-4 right-4 → أسفل اليمين
```

### **تغيير نص الزر:**

```jsx
// Toast - السطر ~150

<button ...>
    <span>عرض التفاصيل</span> {/* غيّر النص هنا */}
    <ArrowLeft size={16} />
</button>
```

### **تغيير لون الزر:**

```jsx
// Toast - السطر ~145

className={`
    ...
    ${toast.priority >= 5 
        ? 'bg-red-500 hover:bg-red-600 text-white'  // أولوية عالية
        : 'bg-brand hover:bg-brand/90 text-white'}   // عادي
`}
```
